# Cenpo

A handwritten clone of my vibe coded
[tenpo](https://github.com/JakeI/conf/tree/master/tenpo)
that is in turn a clone of
[Ultimate Time Tracker (utt)](https://github.com/larose/utt).

## Core idea

Use nothing but the c std library.

## This is a work in progress

- So far it's mostly an args parser.
- But csv reporting already works.
- Next up should be filtering.
